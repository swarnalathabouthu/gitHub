****Commands Used****

*First Create a repository in GitHub without "add ReadMe file" and copy the remote path.

> git -v
To check the git version.

> git init
Used to start Initilazation, it helps to create a git repository in GitHub.

> git add
Used to add a single file in GitHub.

> git add .
Used to add a all files in folder in GitHub.

> git commit
Used to save the changes and save the added files in GitHub.

> git remote add origin "https://github.com/swarnalathabouthu/Gittaskone.git"
Giving a connection to send laptop files to remote(repository) in GitHub by giving exact location(repository path) .

> git branch -M main
Create a new Main branch
> git push origin main
Used to push the laptop main folder to GitHub main repository.
